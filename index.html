Part 01

LangGraph Enna? Yean Vandhichu?

Traditional LangChain "chains" na, ஒரு straight line — A → B → C, mudinjadhu. Ana real-world workflows ellam straight illa. Doctor patient-a paakum maadhiri — decision edukanum, back-and-forth pesanum, loop varanum.

LangGraph = Graph structure vachu, AI agent-oda "thinking flow"-a build pannurom. Idhu State Machine based — ஒரு graph la nodes (steps) irukum, edges (connections) irukum, adhu ellam oru shared "State" (notebook) vachu communicate pannum.

Real World Analogy — Hospital Flow

Patient admit aana pinnadi enna nadakum-nu paakalam:

  • Nodes = Ovvoru step (Physician, Cardiologist, Surgeon) — oru specific vela pannum function.
  • Edges = Rendu node-ku edaila connection — next step enna nu decide pannum path.
  • Conditional Edges = "Manage with Medicine?" nu question kekurathu maadhiri — decision base pannitu next node select pannuvom.
  • State = Patient-oda full record (notebook) — ovvoru node idha padikalam, update pannalam.

Rendu type edges irukum — Normal Edge (fixed path) & Conditional Edge (function decide pannum).

class RAGState(TypedDict):
    query: str
    chunks: list
    prompt: str
    answer: str

def retrieve(state: RAGState):
    chunks = vector_store.similarity_search(state["query"], k=5)
    return {"chunks": chunks}

def build_prompt(state: RAGState):
    context = "\n".join([c.page_content for c in state["chunks"]])
    prompt = f"Answer ONLY from this context:\n{context}\n\nQuestion: {state['query']}"
    return {"prompt": prompt}

def generate(state: RAGState):
    answer = call_claude(state["prompt"])
    return {"answer": answer}

graph = StateGraph(RAGState)
graph.add_node("retrieve", retrieve)
graph.add_node("build_prompt", build_prompt)
graph.add_node("generate", generate)
graph.set_entry_point("retrieve")
graph.add_edge("retrieve", "build_prompt")
graph.add_edge("build_prompt", "generate")
graph.add_edge("generate", END)
rag_app = graph.compile()

9.3 main.py — Full Code

Idhu single file la State, 3 Nodes (classify / billing / technical / general), Conditional Edge, Graph compile, run loop — ellam irukum.

from typing import TypedDict
from langgraph.graph import StateGraph, END
from langchain_anthropic import ChatAnthropic
from langchain_core.prompts import ChatPromptTemplate
from dotenv import load_dotenv

load_dotenv()
llm = ChatAnthropic(model="claude-sonnet-4-6")

# ---- 1. State: graph full-a share aagura "notebook" ----
class State(TypedDict):
    query: str
    category: str
    answer: str

# ---- 2. Nodes ----
def classify(state: State):
    prompt = ChatPromptTemplate.from_template(
        "Classify this message into ONE word: billing, technical, or general.\nMessage: {query}\nCategory:"
    )
    result = (prompt | llm).invoke({"query": state["query"]})
    return {"category": result.content.strip().lower()}

def billing_node(state: State):
    return {"answer": "Routing you to our billing team. They'll email within 24 hours."}

def technical_node(state: State):
    prompt = ChatPromptTemplate.from_template("Give a short troubleshooting tip for: {query}")
    result = (prompt | llm).invoke({"query": state["query"]})
    return {"answer": result.content}

def general_node(state: State):
    return {"answer": "Thanks for reaching out! Our team will get back to you shortly."}

# ---- 3. Conditional Edge: category base pannitu route pannurathu ----
def route(state: State) -> str:
    if "billing" in state["category"]:
        return "billing"
    elif "technical" in state["category"]:
        return "technical"
    return "general"

# ---- 4. Build the Graph ----
graph = StateGraph(State)
graph.add_node("classify", classify)
graph.add_node("billing", billing_node)
graph.add_node("technical", technical_node)
graph.add_node("general", general_node)

graph.set_entry_point("classify")
graph.add_conditional_edges("classify", route, {
    "billing": "billing",
    "technical": "technical",
    "general": "general",
})
graph.add_edge("billing", END)
graph.add_edge("technical", END)
graph.add_edge("general", END)

app = graph.compile()

# ---- 5. Run it ----
if __name__ == "__main__":
    while True:
        query = input("\nCustomer message (or 'quit'): ")
        if query.lower() == "quit":
            break
        result = app.invoke({"query": query})
        print(f"Category: {result['category']}")
        print(f"Answer:   {result['answer']}")
